## 작업 회고
- HTML작성하면서 테스트하기 어렵다.
- 어드민부분에서 신경써야되거나 추가되야하는 것들이 많다보니 헷갈리는 경우가 많다
- 타임리프, 쿼리 스트링을 사용하며 메인 페이지의 흐름을 생각할때 혼자 작업하면 어떻게 작업이 진행 되어있는지 쉽게 알수 있지만 팀원들과 작업을 하다보니 코드 하나하나 보면서 연결하는 작업이 오래걸린다.
  팀장으로써 팀 전체적인 작업 흐름을 확인을 해야하는데 흐름을 완벽하게 이해하지 못해서 문제가 발생하니 좀더 노력을 해야겠고 팀원들에게 할당할 작업들을 계속 생각해야겠다.
- 이메일 인증에 오류가 발생했고 그걸 해결하기위해 팀원들과 여러 시도를 거쳤으나 해결되지않았음. 다른 방법을 더 고민해봐야할 것 같음.컨트롤러test을 완료하고 게시물조회html을 작성시작함.
- 컨트롤러 테스트 시 테스트 유저 생성 및 게시글 생성의 필요성을 느끼게 되었다.
