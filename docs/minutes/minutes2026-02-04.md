# 2차 프로젝트 1팀 회의록
- 날짜 2026년 02월 04일
- 팀명 : OneBoard
- 참여자 : 정확,김재웅,박민성,최서윤,김정인
- 파일명 : community
## 오전회의 내용
- 커뮤니티 주제 정하기
- 맛집에 관한 주제 : 걸어서 맛집으로
- 기능 명세서 작성 하기
- 피그마 화면설계 보안 및 수정
- API 기능 명세서 작성 시작하기
## 작업 목표
- 기능 명세서 완성하기
- 피그마 화면 설계 90% 완료하기
- API 기능 명세서 50% 작성하기
## 작업 회고
- 화면 설계시 전체적인 페이지 흐름을 처음 작성하다보니 미숙한 부분이 많이 보임
- 기능 명세서 작성도 미숙한 부분이 많아 아직 감 잡는 게 어려워서 작업 할때 더디게 시작하는 느낌이 있다.
- 화면 설계중 ERD 설계 에 추가할 부분이 있어 왔다 갔다 반복잡이 많이 일어나 시간 낭비가 눈에 보인다 이로인해 처음부터 꼼꼼히 작성해야된다
## 내일목표
- 기능명세서 작성 완료 후 컴펌 받고 통과하기
- 피그마 화면 설계서 에서 관리자 대시보드 빠트린부분 보충하고 그에 맞는 ERD 수정이 이뤄저야 하면 이루어 져야한다
- 모든 컴펌이 통과 되면 역할 분담후 기본적인 코드 작성 시작하기
## 팀 규칙
- 팀원간 싸우지 않기
- 안된다 외치기 금지
- 항상 웃으면서 작업하기
- 각자 작업과정 작업회고 적어두기
- 문제점 및 오류 등 명확하게 작성하여 제출
- 문제가 생겨서 도움요청시 바로바로 도와주기
- 항상 밥은 맛있는거 먹고오기
- 3시30분 부터 내일목표 천천히 생각해보기
